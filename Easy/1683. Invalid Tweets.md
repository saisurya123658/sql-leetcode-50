# 📌 Problem Statement

Table: `Tweets`

| Column Name | Type |
|-------------|------|
| tweet_id | int |
| content | varchar |

### 📝 Description

- `tweet_id` is the primary key.
- `content` contains:
  - Alphanumeric characters
  - Spaces
  - `!`
- Each row represents a tweet in a social media application.

---

# 🎯 Task

Find the IDs of invalid tweets.

A tweet is considered **invalid** if:

✅ The number of characters in `content` is **greater than 15**

Return the result table in **any order**.

---

# 💻 My SQL Solution

```sql
SELECT tweet_id
FROM Tweets
WHERE CHAR_LENGTH(content) > 15;
```

---

# 🧠 Explanation of My Approach

To solve this problem, I filtered tweets based on the length of their content.

---

## Step 1️⃣ Select Tweet IDs

```sql
SELECT tweet_id
```

This returns only the tweet IDs.

---

## Step 2️⃣ Read Data From Table

```sql
FROM Tweets
```

The data is fetched from the `Tweets` table.

---

## Step 3️⃣ Check Tweet Length

```sql
WHERE CHAR_LENGTH(content) > 15;
```

This checks whether the number of characters in the tweet content exceeds `15`.

If true, the tweet is invalid.

# ⚠ Important SQL Concept

## What Does `CHAR_LENGTH()` Do?

```sql
CHAR_LENGTH(content)
```

Returns the total number of characters in the string.

Example:

```sql
CHAR_LENGTH('Hello')
```

Result:

```text
5
```

Spaces are also counted as characters.
