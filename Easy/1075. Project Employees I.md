# рџЋЇ Task

For each project, calculate:

```text
Average Experience Years
=
Sum of Employee Experience Years
```

вњ… Return `average_years`

# рџ’» My SQL Solution

```sql
SELECT
    p.project_id,
    ROUND(AVG(e.experience_years), 2) AS average_years
FROM Project p
JOIN Employee e
ON p.employee_id = e.employee_id
GROUP BY p.project_id;
```

---

# рџ§  Explanation of My Approach

The employee experience is stored in the `Employee` table, while project assignments are stored in the `Project` table.

To calculate the average experience for each project:

1. Join both tables.
2. Get employees belonging to each project.
3. Calculate the average experience.
4. Round the result.

# вљ  Important SQL Concept

### Why AVG()?

Instead of manually writing:

```sql
SUM(experience_years)
/ COUNT(*)
```

SQL provides:

```sql
AVG(experience_years)
```

which automatically calculates the average.

---

### Why JOIN?

The Project table only contains employee IDs.

The Employee table contains experience years.

To access both pieces of information, we need:

```sql
JOIN
```
