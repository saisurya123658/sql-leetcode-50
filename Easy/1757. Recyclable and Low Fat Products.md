# 📌 Problem Statement

Table: `Products`

| Column Name | Type |
|-------------|------|
| product_id  | int |
| low_fats    | enum |
| recyclable  | enum |

### 📝 Description

- `product_id` is the primary key.
- `low_fats` is an ENUM with values `('Y', 'N')`
  - `Y` → Product is low fat
  - `N` → Product is not low fat
- `recyclable` is an ENUM with values `('Y', 'N')`
  - `Y` → Product is recyclable
  - `N` → Product is not recyclable

---

# 🎯 Task

Write a SQL query to find the IDs of products that are:

✅ Low fat  
✅ Recyclable  

Return the result table in **any order**.

---

# 💻 My SQL Solution

```sql
SELECT product_id
FROM Products
WHERE low_fats = 'Y'
AND recyclable = 'Y';
```

---

# 🧠 Explanation of My Approach

To solve this problem, I filtered the rows using the `WHERE` clause.

### Step-by-Step:

### 1️⃣ Select Required Column

```sql
SELECT product_id
```

This returns only the product IDs.

---

### 2️⃣ Read Data From Table

```sql
FROM Products
```

The data is taken from the `Products` table.

---

### 3️⃣ Apply Conditions

```sql
WHERE low_fats = 'Y'
AND recyclable = 'Y';
```

This checks:

- Product must be low fat (`Y`)
- Product must be recyclable (`Y`)

The `AND` operator ensures both conditions are true.
