# 📌 Problem Statement

### Delivery Table

| Column Name | Type |
|------------|------|
| delivery_id | int |
| customer_id | int |
| order_date | date |
| customer_pref_delivery_date | date |

### 📝 Description

- Each row represents a food order.
- If:

```text
order_date = customer_pref_delivery_date
```

the order is called **Immediate**.

Otherwise, it is **Scheduled**.

- Every customer has exactly one first order.
- The first order is the order with the earliest `order_date`.

---

# 🎯 Task

Find the percentage of customers whose **first order** was an **Immediate Order**.

Return:

| immediate_percentage |

Rounded to **2 decimal places**.

---

# 💻 My SQL Solution

```sql
SELECT
    ROUND(
        100.0 *
        SUM(
            CASE
                WHEN order_date = customer_pref_delivery_date
                THEN 1
                ELSE 0
            END
        ) / COUNT(*),
        2
    ) AS immediate_percentage
FROM Delivery
WHERE (customer_id, order_date) IN
(
    SELECT customer_id, MIN(order_date)
    FROM Delivery
    GROUP BY customer_id
);
```

---

# 🧠 Explanation of My Approach

The problem only asks about:

> The first order of every customer

So first:

1. Find each customer's earliest order.
2. Check whether that order is immediate.
3. Calculate the percentage.

# Step 5️⃣ Calculate Percentage

Formula:

\[
\text{Immediate Percentage}
=
\frac{\text{Immediate First Orders}}
{\text{Total First Orders}}
\times 100
\]

\[

50
\]

# ⚠ Important SQL Trick

Instead of writing:

```sql
COUNT(CASE WHEN ... END)
```

You can convert conditions into:

```sql
1 → True
0 → False
```

and use:

```sql
SUM()
```

to count matching rows.
